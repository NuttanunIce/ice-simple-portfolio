# Portfolio Website Development Plan

## Objective
Build a modern, dark-themed, data analyst portfolio website hosted on GitHub Pages.

## Requirements
1. **Design:** Modern dark theme, JetBrains Mono font.
2. **Hero:** "Nuttanun Ice" | Data Analyst.
3. **Projects:** 3 highlighted data projects.
4. **Contact:** Clear contact details/links.

## Implementation Steps

### Phase 1: Setup & Environment
1. Initialize a new local directory.
2. Setup basic HTML/CSS structure.
3. Configure Git repository.

### Phase 2: Design Implementation
1. Apply dark theme CSS.
2. Integrate JetBrains Mono font.
3. Implement Hero section.

### Phase 3: Content Integration
1. Add placeholder structure for 3 data projects.
2. Add contact information section.

### Phase 4: Deployment
1. Configure GitHub Pages settings.
2. Push code to `main` branch.

## Verification
- Validate dark theme and font loading.
- Check responsiveness on various screen sizes.
- Verify project and contact link functionality.
- Ensure site is live on GitHub Pages.
